# 村域图斑切分学习路径：总索引与阅读顺序

这套文档已经按“由浅入深”的学习路径重排。目标不是先把所有方案都看完，而是先建立正确模型，再补最小基础，再逐步进入代码实现与方案比较。

## 一、整套材料怎么分层

### 1. 问题建模与最优标准

路径：

`01-问题建模与最优标准/村域图斑重分组：邻接图上的连通平衡划分.md`

这一层解决的问题：

- 这个问题本质上到底是不是“几何切割”
- 为什么“不能切图斑”会把问题改写成“邻接图上的连通平衡划分”
- 什么叫硬约束，什么叫软目标
- 怎么定义“最优切分方案”

如果你只读一篇来把问题看对，就先读这一篇。

### 2. 空间基准方案

路径：

`02-空间基准方案/空间划分基准：从KD树、四叉树到连通性修复.md`

`02-空间基准方案/KD树图斑分组基准：纯空间切分与连通拆分.md`

这一层的作用不是给最终答案，而是建立一个“空间优先”的基准视角。

你会看到：

- KD 树 / 四叉树为什么容易上手
- 为什么空间接近不等于拓扑正确
- 为什么“连通性修复”一加进来，问题就开始向图划分靠拢

### 3. 图论基础与最小练习

路径：

`03-图论基础与最小练习/图斑重分组的图论入门：从业务映射到最小知识树.md`

`03-图论基础与最小练习/图论入门第一课：用Java实现连通分量.md`

这一层只补“够用的图论”：

- 邻接、路径、连通分量
- DFS / BFS
- 树、生成树
- 为什么这些概念和你的图斑切分是一一对应的

这部分是你真正开始自己写代码的入口。

### 4. Java 实现与示例工程

路径：

`04-Java实现与示例工程/村域图斑空间分区方案与Java示例/村域图斑空间分区方案：思路总览与实现说明.md`

`04-Java实现与示例工程/村域图斑空间分区方案与Java示例/示例代码/`

这一层给你第一套完整的 Java 工程骨架，重点是：

- JTS 在这个问题里负责什么
- 一个完整的切分器、验证器、测试数据生成器应该怎么组织
- 如何从说明文档映射到代码文件

### 5. 方案对比与可视化

路径：

`05-方案对比与可视化/村域图斑分区方案对比与可视化/`

`05-方案对比与可视化/三种村域图斑切分方案对比与可视化/`

这一层是横向比较：

- 拓扑安全坐标二分
- 多层级图划分
- 最大生成树切分

这里最适合你建立“不同方案各自适合什么”的判断力。

### 6. 最大生成树方案专项

路径：

`06-最大生成树方案专项/最大生成树学习路径与分析/`

这一层是进阶主线。我建议你把它看成“最值得内化的一套方案”。

它串起来的知识包括：

- 并查集
- 生成树
- Kruskal
- 子树大小
- 递归平衡切树

## 四、每一阶段你应该自己写什么

### 阶段 A：补最小基础

自己写：

- `Graph`
- `addUndirectedEdge`
- `connectedComponents`
- `isConnectedSubset`

### 阶段 B：进入树

自己写：

- `UnionFind`
- `Kruskal` 求最大生成树
- 树的邻接表
- `computeSubtreeSizes`

### 阶段 C：进入切分

自己写：

- 在树上枚举切边
- 计算两侧大小
- 选择最均衡的边
- 对超过阈值的子树递归切分

### 阶段 D：接入 GIS

自己写：

- 基于 JTS 的邻接图构建
- 共边长度计算
- 分组结果 `union` 成区域多边形
- 基本验证器

## 五、怎么用这套材料

不要一开始就读所有代码。

更高效的方式是：

1. 先读概念文档
2. 自己写最小版本
3. 再看示例代码
4. 再看对比与可视化
5. 最后整理成自己的版本

这套材料最适合的使用方式不是“照抄”，而是“先自己做，再回来对照”。
